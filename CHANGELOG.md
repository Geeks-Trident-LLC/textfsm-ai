# Changelog

All notable changes to **textfsm-ai** are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and this project adheres to [Semantic Versioning](https://semver.org/).

---

## [0.1.0] - 2026-05-29
### Added
- Initial public release of **textfsm-ai**
- Multi-provider routing (OpenAI, Claude, Gemini, DeepSeek)
- QuotaManager with daily/monthly token tracking
- Provider abstraction layer with consistent interfaces
- CLI entry point for template generation
- YAML-based configuration loader
- Unit test suite for providers, router, and quota logic
- PyPI-ready packaging with classifiers, metadata, and extras
- GitHub Actions workflows for CI and publishing
- Documentation starter (MkDocs structure)
